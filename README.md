# Temperaturdaten
Temperaturdaten ins Internet mit Auswertung (ESP8266)

Das Endziel des Projektes ist eine autarke Messstation mit Hilfe des ESP8266-01 im Garten zu betreiben, welches seine Temperaturdaten auf eine private Internetseite ablegt. Diese Daten sollten den aktuellen Wert anzeigen und zu statistischen Zwecken grafisch ausgegeben werden kÃ¶nnen.
